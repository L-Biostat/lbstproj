# Build a report file name with date, version, and optional counter and ext

Build a report file name with date, version, and optional counter and
ext

## Usage

``` r
build_report_name(extension, counter = NULL)
```

## Arguments

- extension:

  The file extension for the report (e.g., "html", "pdf").

- counter:

  An optional counter to append to the file name for uniqueness. If
  `NULL`, no counter is added.
